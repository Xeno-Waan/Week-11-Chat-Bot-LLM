# Folder Dokumen RAG

Letakkan berkas materi kuliah Anda di sini agar diindeks secara otomatis oleh chatbot.

### Format Berkas yang Didukung:
1. **`.txt`** — Berkas teks biasa.
2. **`.md`** — Catatan Markdown.
3. **`.pdf`** — Berkas dokumen PDF.

*Catatan: Jika folder ini kosong, chatbot secara otomatis akan menggunakan data simulasi bawaan.*
